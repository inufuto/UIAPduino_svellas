# UIAPduino+SSD1306版 Svellas mini

並んだパネル上の通路を通って、星を全て取るとクリアーです。  
パネルを動かすことで、通路を繋ぎ変えたり離れたパネルの間を渡ることができます。

![](rec.jpg)

## 操作

|操作|割り当て|
|-|-|
|移動|UP, DOWN, LEFT, RIGHT|
|パネルの移動|矢印を踏んでACT|
